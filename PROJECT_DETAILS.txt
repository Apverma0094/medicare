===============================================
                 API USAGE
===============================================

✅ INTERNAL APIs (Self-created):
--------------------------------

🛒 CART MANAGEMENT APIs:
• GET  /cart/count          → Get cart items count
• POST /cart/add            → Add item to cart
• PATCH /cart/update        → Update cart item quantity
• DELETE /cart/remove/{id}  → Remove item from cart
• DELETE /cart/clear        → Clear entire cart

📦 ORDER MANAGEMENT APIs:
• GET  /orders/count        → Get active orders count
• POST /place-order         → Place new order
• GET  /my-orders          → Get user order history
• GET  /orders/{id}/track   → Track specific order

🔔 ADMIN APIs:
• GET  /admin/notifications/count    → Notification count
• GET  /admin/notifications/latest   → Latest notifications
• POST /admin/notifications/{id}/read → Mark notification as read
• POST /admin/notifications/read-all  → Mark all as read

✅ EXTERNAL APIs:
-----------------

💳 STRIPE PAYMENT API:
• Stripe Checkout Sessions
• Payment processing
• Webhook handling
• Success/Cancel redirects

===============================================
        FRONTEND-BACKEND COMMUNICATION
===============================================

🔄 AJAX COMMUNICATION PATTERN:
------------------------------
Frontend (JavaScript) 
     ↓ fetch() request
Backend (Laravel Controller)
     ↓ JSON response  
Frontend (Update UI dynamically)

📡 REAL AJAX EXAMPLES:

1. CART COUNT UPDATE:
```javascript
async function loadCartCount() {
    const response = await fetch('/cart/count');
    const data = await response.json();
    updateCartCount(data.count);
}
```

2. ADD TO CART:
```javascript
async function addToCart(medicineId) {
    const response = await fetch('/cart/add', {
        method: 'POST',
        headers: {
            'Content-Type': 'application/json',
            'X-CSRF-TOKEN': csrfToken
        },
        body: JSON.stringify({
            medicine_id: medicineId,
            quantity: 1
        })
    });
}
```

3. ORDER COUNT:
```javascript
async function loadOrderCount() {
    const response = await fetch('/orders/count');
    const data = await response.json();
    updateOrderCount(data.count);
}
```
